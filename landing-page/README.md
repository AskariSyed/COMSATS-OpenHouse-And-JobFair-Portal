# jfair.tech Landing Site (React)

This folder contains the React + Vite + TypeScript landing website for `jfair.tech`.

## Stack
- React 18
- TypeScript
- Vite
- React Router
- Tailwind CSS
- Framer Motion

## Routes
- `/` Home overview
- `/student` Student portal details
- `/company` Company portal details
- `/admin` Admin portal details
- `/features` Feature matrix with filters
- `/how-to-use` Role-based onboarding steps

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Branding
- Logo and favicon: `public/assets/LogoWithoutBg.png`
